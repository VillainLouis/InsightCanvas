# 安装
```shell
snap install nvim --classic
```



一下配置都是基于Lazy.vim

# neovim基本配置

配置文件路径：`~/.config/nvim/`

# neovim自己常用命令

打开当前文件夹目录：<space> + e

切换目录和文件光标：<ctrl> + h | <ctrl + l>

调整目录显示和文件显示的宽度：<ctrl> + 左右方向键

已经打开的文件之间切换：H | L


# 插件管理

## LSP

Lazy.vim使用Manson.vim管理LSP

进入插件管理的GUI：`:Manson`
